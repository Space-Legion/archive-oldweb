Ventoy for linux



# 1 Format your pendrive


# 2 Download latest release of Ventoy from here


# 3 Download ventoy tar.gz file for linux.


# 4 Now extract the zip and open up terminal.


# 5 ls (to view all the files in a directory)


# 6 lsblk (To find your disk name )


# 7 sudo ./Ventoy2Disk.sh (To run the file)


If that doesnt work try using chmod +x Ventoy2Disk.sh and try #7 step.


# 8 sudo ./Ventoy2Disk.sh -i /dev/sdX (Example: sudo ./Ventoy2Disk.sh -i /dev/sda)


It takes less than 30s.


If that doesnt help dimount the pendrive and try it again or


Now copy the ISO files to the pendrive.
