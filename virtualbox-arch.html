Virtual-Box Install Arch


sudo pacman -S virtualbox

And select the package virtualbox-host-modules-arch, ie option 2 while installing

sudo modprobe vboxdrv

sudo nano /etc/modules-load.d/virtualbox.conf (type vboxdrv then save and exit, So we don't have to start it manually )

yay virtualbox-ext-oracle (or go to virtualbox's and download the extension pack https://www.virtualbox.org/wiki/Downloads)

sudo gpasswd -a $USER vboxusers (This is necessary in order to fully access the features provided by VirtualBox, including the ability to use USB devices in a Guest operating system.)
